# 📱 Atividade de Reposição - Harry Potter

## 🔗 API utilizada

Este projeto utiliza a API pública de personagens de Harry Potter:
https://hp-api.onrender.com/api/characters

---

## 🚀 Como rodar o projeto

1. Acesse o Expo Snack ou rode localmente com Expo
2. Caso rode localmente:

```bash
npm install
npx expo start
```

3. Escaneie o QR Code com o celular ou abra no emulador

* Esse projeto foi feito usando o Expo Snack(Não estava carregando o npm install no meu visual studio code) caso prefira abrir o código por lá: https://snack.expo.dev/@tatizabaneh/api---hp---mobile
---

## ⚙️ Funcionalidades

*  Busca de personagens por nome ou nomes alternativos
*  Exibição da imagem do personagem
*  Informações do ator
*  Dados pessoais (espécie, gênero, características físicas, status de vida, etc..)
*  Mudança de cor de fundo conforme a casa de Hogwarts(Grifinoria: vermelho, Sonserina: verde, etc)
*  Fundo especial para personagens que são staff(castelo de hogwarts)
*  Indicador de carregamento (loading)
*  Tratamento de erro quando personagem não é encontrado
